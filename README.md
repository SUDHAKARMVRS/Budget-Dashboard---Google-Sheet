# Budget Dashboard (Google Sheets)

An interactive **Budget Dashboard built using Google Sheets** to track income, bills, expenses, savings, and debt with dynamic charts and breakdown tables.

## Project Overview

This dashboard helps users monitor their monthly finances in one place.  
It provides visual insights into spending patterns using charts and automated calculations.

The dashboard is designed to be **interactive, simple, and easy to update**.

---

## Features

- Interactive **Budget vs Actual comparison**
- Dynamic **Bills, Expenses, and Debt tracking**
- **Tick-box controlled charts**
- Automatic **Spending Breakdown tables**
- **Pie and Donut charts visualization**
- **Currency selector** (£, $, €)
- **Left to Spend indicator**
- Printable **A4 dashboard layout**

---

## Dashboard Sections

### 1. Overview
- Start Date
- End Date
- Currency Selection
- Start Balance

### 2. Cash Flow
Tracks:
- Income
- Bills
- Expenses
- Savings
- Debt
- Remaining balance

### 3. Bills / Expenses / Debt Tables
Each section includes:
- Category
- Budget
- Actual
- Tick boxes to include/exclude items

Charts automatically update based on selections.

### 4. Spending Breakdown
Displays:
- Category
- Amount
- Percentage of total spending

### 5. Visualizations
- Budget vs Actual bar chart
- Bills spending pie chart
- Expenses spending donut chart
- Debt distribution pie chart

---

## Tools Used

- Google Sheets
- Data validation
- Checkbox controls
- Formulas:
  - `SUM`
  - `IF`
  - `FILTER`
  - `SORT`
  - `VLOOKUP`

---

## How to Use

1. Enter **income and starting balance**
2. Add **bills, expenses, and debt values**
3. Tick categories you want to include
4. Charts and breakdown tables update automatically
